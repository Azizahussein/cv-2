Grade Calculator Site

I developed a full-stack Grade Calculator project that required user authentication for secure access. The application included a display page where users could view their courses, with functionality to edit or delete courses as needed. I built the front end using React and TypeScript, creating an interactive and responsive interface, while the backend was implemented with MongoDB, enabling efficient storage and management of course data. This project demonstrated seamless integration between the front-end and back-end, ensuring a smooth and user-friendly experience.

A Trio of Brain, Artificial General Intelligence, and Neuromorphic Computing

Research 1: This ongoing research paper, set to be completed and published on May 13, explores the intersection of neuroscience, artificial general intelligence (AGI), and neuromorphic computing. Currently, I am analyzing human stress responses through functional MRI (fMRI) data, combined with large language models (LLMs) to interpret cognitive and emotional patterns under stress. The project aims to draw parallels between biological brain processes and computational models, providing insights that could contribute to the development of more human-like AGI systems. By integrating AI-driven analysis with neuroimaging, this research bridges the gap between brain function and machine intelligence.

Music Summary Site

I developed a full-stack music-sharing website that connects to the Spotify API to display users’ top songs of the week. The application features Spotify authentication for secure, personalized access and updates automatically each week to keep content fresh. Users can share their favorite tracks with friends, encouraging social interaction around music. Built with React and JavaScript on the front end, and MongoDB on the back end, the site delivers a seamless and engaging user experience, styled with Tailwind CSS for a clean, modern interface.
